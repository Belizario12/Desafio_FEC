# Gerenciador de Notebooks

Sistema de gerenciamento de empréstimos de notebooks desenvolvido em Ruby on Rails.

## 🚀 Requisitos

- Ruby 3.4.x 
- Rails 7.x
- PostgreSQL
- Bundler

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/Belizario12/Desafio_FEC.git
cd Desafio_FEC
bundle install
rails db:create
rails db:migrate
rails server
